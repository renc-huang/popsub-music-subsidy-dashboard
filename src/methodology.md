# 方法論

## 資料來源

### 補助軌（P1）

- **來源**：文化部獎補助資訊網（grants.moc.gov.tw）
- **範圍**：流行音樂產業組（unitKeyWord=49）所有獎補助名單公告
- **涵蓋年度**：107–114 年度
- **採集方式**：Playwright 自動化爬取（兩層架構：清單頁 → 個案明細頁）
- **粒度**：每筆 = 一個獲補助者的一個計畫案

### 採購軌（P3）

- **來源**：中華民國政府電子採購網，經 g0v pcc-api（pcc-api.openfun.app）間接取得
- **範圍**：文化部影視及流行音樂產業局（unit_id=A.25.2），篩選 job_number 以 D 開頭者（流行音樂產業組）
- **涵蓋年度**：101–115 年度
- **採集方式**：Playwright 經由 API 取得（繞過 Cloudflare 防護），含三層查詢（清單 → 標案明細 → 得標廠商）
- **去重規則**：以 job_number 為唯一鍵，每標案僅保留一筆（決標公告優先，無決標則保留最新）
- **粒度**：每筆 = 一個 unique 標案

### 預決算（P2）

- **來源**：文化部影視及流行音樂產業局官網（bamid.gov.tw）行政公開資訊
- **涵蓋年度**：102–114 年度
- **採集方式**：pdfplumber 解析法定預算與決算審定本 PDF；110 年度使用 XML 結構化資料
- **目標項目**：「流行音樂產業輔導」（預算科目編號 5361201200）

## 已知限制

1. **「流行音樂」定義不統一**：補助平台以申請單位的主業認定，採購網以標案品項關鍵字搜尋，兩者母群體定義不同
2. **採購軌以 D 前綴篩選**：此為流行音樂產業組的標案編號慣例，但無法排除部分跨組案件可能使用其他前綴
3. **廠商名稱未標準化**：同一實體可能以不同名稱出現於兩軌資料中，目前尚未建立統一對照表
4. **年度口徑差異**：補助年度以核定年度計算，採購以決標年度計算，兩者未必對應同一曆年的實際支出
5. **預決算解析品質**：部分早期 PDF 為掃描件或特殊字型編碼，已以人工方式補錄
